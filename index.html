HTML

HTML (Hypertext Markup Language) is the standard markup language used for creating and structuring the content of web pages and applications. It provides a set of tags and elements that define the structure, layout, and formatting of the content displayed on a web page.

Level: ⭐⭐⭐⭐

CSS

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation and visual appearance of a document written in HTML or XML (including XHTML). It allows web developers to define how the elements of a web page should be displayed, including layout, colors, fonts, and other design aspects.

Level: ⭐⭐⭐⭐

JavaScript

JavaScript is a high-level programming language that is primarily used for adding interactivity and dynamic functionality to web pages. It allows web developers to create dynamic content, respond to user actions, manipulate and update page elements, and interact with web APIs (Application Programming Interfaces).

Level: ⭐⭐⭐⭐

Express JS

Express.js, or simply Express, is a minimalistic and flexible web application framework for Node.js. It provides a set of robust features and utilities for building web applications and APIs. Express.js is widely used and highly popular among developers due to its simplicity, versatility, and scalability.

Level: ⭐⭐⭐

Node JS

Node.js is an open-source, server-side JavaScript runtime environment built on Chrome's V8 JavaScript engine. It allows developers to execute JavaScript code on the server, enabling them to build scalable and high-performance web applications. Unlike traditional JavaScript, which runs in the browser, Node.js runs on the server and provides a runtime environment for executing JavaScript code outside of a browser context.

Level: ⭐⭐⭐

Bootstrap

Bootstrap is a popular open-source front-end framework used for designing responsive and mobile-first web pages and web applications. It provides a collection of pre-built HTML, CSS, and JavaScript components, as well as a grid system and styling templates, which simplify the process of creating visually appealing and responsive user interfaces.

Level: ⭐⭐⭐

Mongo DB

MongoDB is a popular open-source, NoSQL (non-relational) database management system. It is designed to store and manage large volumes of data in a flexible, scalable, and document-oriented manner. MongoDB departs from the traditional relational database model and instead uses a JSON-like document format called BSON (Binary JSON) to store data.

Level: ⭐⭐⭐

React JS

React.js, commonly referred to as React, is an open-source JavaScript library for building user interfaces. It was developed by Facebook and is widely used for creating interactive and reusable UI components in web applications.

Level: ⭐⭐⭐
